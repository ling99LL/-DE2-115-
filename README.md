# 🚦 基于 Verilog HDL 的交通灯控制系统

> 本项目是《数字集成电路课程设计》的实践成果，使用 **Verilog HDL** 在 **FPGA（DE2-115 开发板）** 上实现一个功能完整的十字路口交通灯控制系统。

---

## ✨ 功能特点

- **自动模式**：  
  每个方向按 `绿灯30秒 → 黄灯5秒 → 红灯30秒` 循环切换。
- **紧急模式**：  
  按键触发后，所有方向红灯亮起，暂停通行。
- **夜间模式**：  
  所有方向黄灯以 1Hz 频率闪烁，提示减速慢行。
- **手动模式**：  
  通过按键手动切换当前灯态（每次按一下切换一次）。
- **多端显示**：
  - **LED**：6 个 LED 模拟红/黄/绿信号灯；
  - **数码管**：2 位 7 段数码管实时显示倒计时；
  - **LCD1602**：显示当前方向（East/West）与灯色（Red/Yellow/Green）。

---

## 🧠 核心模块

| 模块名           | 功能说明 |
|------------------|--------|
| `clk`            | 两级分频器：50MHz → 2Hz → 1Hz，提供系统基准时钟 |
| `key_debounce`   | 按键消抖模块，输出稳定脉冲与电平信号 |
| `traffic_light`  | 主状态机，控制交通灯逻辑与多模式切换 |
| `seg4_7`         | BCD 转 7 段数码管驱动（组合逻辑，无延迟） |
| `lcd1602`        | LCD1602 显示驱动，支持字符动态刷新 |
